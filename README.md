# Iris-tableau-dashboard
The Iris Tableau Dashboard is an interactive data visualization project built using Tableau to analyze the famous Iris flower dataset. The dashboard explores variations in sepal and petal measurements across three Iris species through statistical summaries, scatter plots, box plots, histograms, and KPI cards. It helps users understand species characteristics, compare distributions, and identify patterns using interactive visualizations.
# Key Insights & Features
🌸 The dataset contains 150 flower records across 3 Iris species: Setosa, Versicolor, and Virginica.
📊 KPI cards provide a quick summary of Total Records, Species Count, Average Sepal Length, and Average Petal Length.
📈 Species Count chart shows an equal distribution of 50 records for each species.
🌼 Sepal Scatter Plot visualizes relationships between sepal length and sepal width for species comparison.
🌺 Petal Scatter Plot clearly separates the three species based on petal measurements.
📦 Box Plot highlights the distribution, spread, and potential outliers of sepal width among species.
📉 Histogram displays the frequency distribution of petal lengths.
📋 Average Metrics Table compares average sepal and petal dimensions across all species.
🎯 Interactive filtering enables users to explore species-specific insights.
